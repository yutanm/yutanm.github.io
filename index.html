使用hexo+github搭建博客

本文配置环境:

Windows 10
Git 2.21
Node.js 10.15.3
npm 6.41

你所需要的：

上方的配置环境
一个Github账号(免费)

  • Hexo

1 .下载

在cmd中使用

install -g hexo-cli

注意:如果速度太慢可以将npm切换到淘宝源

npm install -g cnpm --registry==https://registry.npm.taobao.org

等待下载完成

2 . 使用

建立一个blog文件夹，用来储存博客数据
cd blog文件夹
执行初始化命令

hexo init

3 .启动

hexo s
这时已经启动了，可以使用

http://localhost:4000

进行本地访问

远端部署

   在前文，我们已经将hexo博客部署到本地了，接下来就是将博客部署到远端
本文使用Github仓库来托管。

1.在github上创建一个仓库,

仓库名为 用户名+github.io (此步极为重要)

2.下载部署插件

在cmd中输入
cnpm install --save hexo-deployer-git
会有一个警告，无视即可

3.修改配置文件

修改blog文件夹中的config.yml文件

注意：

blog/config.yml文件为站点配置文件，而blog/themes/config.yml文件为主题配置文件，别搞混了

找到deploy项，修改: 

deploy:   
    type: git
    repo: https://github.com/youname/youname.github.io.git   #仓库的ssh链接
    branch: master  #可加可不加，默认也为master

4.设置Git账户

打开cmd

设置邮箱

git config --global user.email "you@example.com"

设置用户

git config --global user.name "Your Name"

5.远端部署

清理：

hexo clean

生成博客文件：

hexo g

推送到远端,

//中途会提示登陆github
hexo d

至此，一个托管在github的个人博客就完成了，你现在可以使用http:仓库名来访问博客了。
